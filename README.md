# Cat-vs-Dog-Image-Classifier-
A deep learning web application that classifies uploaded images as Cat or Dog using PyTorch, TorchVision, and Flask. The project includes a CNN training pipeline and a user-friendly web interface for real-time image prediction. It demonstrates the complete workflow from model training to deployment.

* Features

Upload an image through a Flask web interface.
Real-time Cat vs Dog image classification.
Image preprocessing with TorchVision transforms.
Deep learning model built with PyTorch.
Supports JPG, JPEG, PNG, WEBP, and BMP images.

* Tech Stack

Python
PyTorch
TorchVision
Flask
Pillow (PIL)
